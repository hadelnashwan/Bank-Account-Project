# Bank Management System – C++ Project

This project is a Bank Management System developed in C++ as part of my learning journey in Algorithms, Problem Solving, and Data Structures.

The system is organized into multiple screens and classes to keep the program structured and easy to manage.

## Features

### Client Management
- Add new client
- Find client
- Update client
- Delete client
- Clients list

### Transactions
- Deposit
- Withdraw
- Transfer
- Transfer logs
- Total balances

### User Management
- Add user
- Delete user
- Find user
- Update user
- Login system

### Currency Management
- List currencies
- Find currency
- Update currency rate
- Currency exchange calculator

## Technologies
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Data Structures

## Purpose
This project was created for educational purposes to practice building structured console applications using C++.
